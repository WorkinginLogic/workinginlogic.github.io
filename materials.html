Stomp Stomps

-Understand the principal functions of a network protocol.
-Understand that both ethernet ports and radio antennas that allow wireless access are layer 1.
-Note that we switch and also bridge at layer 2. Hubs are considered layer 1 due to their lack of addressing.
-Remember the saying, we switch on layer 2, and we route at layer 3.
-An easy trick to remembering that logical ports are on layer 4, is that its in the name of the layer. "TransPORT"
-Understand that ASCII is a way for code from the network layer to be turned into text for the application layer.
-Please note down that our private networks will commonly start with an address of 192. and our public IP will start with 203.
-Remember that Hexidecimal is only numbers 1-9 and letters a-f.
-Note that a receiving node performs decapsulation and the sending node performs encapsulation of data.
-Note that demarcation point is where the telcos cabling enters your premises.
-Understand that in Layer 4, data will be discarded if it has a bad checksum.
-Note that HTTP is found on the final layer of the OSI model, the application layer.
